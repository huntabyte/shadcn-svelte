---
title: June 2026 - RTL Support in the CLI
description: The CLI can now transform components for right-to-left layouts.
date: 2026-06-25
---

RTL support is now built into the shadcn-svelte CLI.

Set `rtl` to `true` in your `components.json` file and the CLI will transform physical Tailwind classes to logical equivalents when adding or updating components. For example, spacing, inset, border, and rounded utilities are mapped from left/right values to start/end values where possible.

```json title="components.json"
{
  "rtl": true
}
```

This release also adds the [Direction](/docs/components/direction) component for providing `ltr` or `rtl` direction context across your app.

See the [RTL documentation](/docs/rtl) for setup instructions.

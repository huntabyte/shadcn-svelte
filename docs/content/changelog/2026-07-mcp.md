title: July 2026 - MCP Server
description: MCP support for browsing, searching, and installing items from the configured shadcn-svelte registry.
date: 2026-07-01
---

The shadcn-svelte MCP server lets AI assistants interact with the registry configured in your project. You can browse available components, search for examples, view registry item details, and get the CLI command to install items directly into your project using natural language.

For example, you can ask an AI assistant to _"Find me a login form example"_ or _"Add the button, dialog and card components to my project."_

Add the MCP server to your project:

```bash
npx shadcn-svelte@latest mcp init
```

See the [MCP server docs](/docs/mcp) for more details.

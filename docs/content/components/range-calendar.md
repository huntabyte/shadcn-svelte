The range calendar is now part of the [Calendar](/docs/components/calendar) component.

Use the `mode` prop to render a range calendar.

```svelte
<Calendar mode="range" />
```

Install and use [Calendar](/docs/components/calendar#installation) for new range calendar implementations.
